What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for HTML and XML documents that represents the structure of a document as a tree of nodes, allowing developers to dynamically access and modify the content, structure, and style of a document.

What is the event delegation in the context of the DOM,and why it is useful?

Event delegation is a technique used in DOM event handling where a single event listener is attached to a parent element, rather than individual child elements. This allows the parent element to capture events that occur on its child elements, and then respond accordingly.Why Event Delegation is Useful:

  • Reduced event listeners
  • Simplified code
  • Dynamic elements
  • Improved maintainability
  • Better scalability
